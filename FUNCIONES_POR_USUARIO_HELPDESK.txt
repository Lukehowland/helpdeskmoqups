================================================================================
                                FLUJO GENERAL DEL SISTEMA
================================================================================

1. USUARIO crea ticket → Sistema asigna ID único → Notifica a AGENTES

2. AGENTE toma ticket → Cambia estado a "En Progreso" → Investiga problema

3. AGENTE responde → USUARIO recibe notificación → USUARIO puede responder

4. Si es complejo → AGENTE escala a EMPRESA → EMPRESA reasigna especialista

5. EMPRESA monitorea métricas → Optimiza procesos → Reporta a ADMIN

6. ADMIN gestiona plataforma → Aprueba empresas → Mantiene sistema estable
